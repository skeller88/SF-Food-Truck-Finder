Food Truck Finder API

This API was built by skeller88 and uses the Mobile Food Facility Permit dataset from Data SF.

It provides the ability to retrieve all of the food trucks that may be within a certain radius of a particular coordinate.

Open up devtools to see your current coordinates. You can use those coordinates to query the API, or choose your own!

Endpoints

GET /foodtrucks

http://sf-food-truck-finder.herokuapp.com/foodtrucks

Parameters
  • Latitude
  • Longitude
  • Radius - Maximum distance from coordinate to search, in miles.
